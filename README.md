How to Build?
-------------

To initialize your local repository using the ColtOS trees, use a 
command like this:

```bash
  repo init -u git://github.com/ColtOS/manifest.git -b cos7.x
```
  
Then to sync up:
----------------

```bash
  repo sync
```
Finally to build:
-----------------

```bash
  . build/envsetup.sh
  brunch <device_name>
```
